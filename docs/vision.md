# Vision

## Mission

DHealth exists to make professional physiotherapy and kinesitherapy accessible, personal, and trustworthy for people in Sofia, Bulgaria. The website is the digital front door — it must communicate competence, warmth, and professionalism within seconds.

## Problem

The current dhealth.bg website is a single-page site with poor mobile experience, no SEO structure, no accessibility, and no clear user journey from discovery to booking. Potential clients searching for therapy in Sofia cannot easily find, evaluate, or contact the clinic.

## Solution

A modern, multi-page, production-grade website that:

1. **Ranks in search** — SEO-optimized pages for every procedure, structured data, fast load times
2. **Builds trust** — Professional design, real testimonials, transparent pricing
3. **Converts visitors** — Clear CTAs, easy booking via form/WhatsApp/Viber, minimal friction
4. **Works everywhere** — Mobile-first responsive, accessible, fast on any device

## Success Metrics

| Metric | Target |
|--------|--------|
| Lighthouse Performance | 90+ |
| Lighthouse Accessibility | 95+ |
| Lighthouse SEO | 95+ |
| First Contentful Paint | < 1.5s |
| Largest Contentful Paint | < 2.5s |
| Mobile usability | 100% (Google Search Console) |
| Pages indexed | All procedure + content pages |

## Non-Goals

- Online payment processing
- User accounts or login
- Real-time scheduling/calendar integration
- Multi-language support (Bulgarian only)
- Blog/content marketing (may be added in a future phase)
- E-commerce checkout for products
